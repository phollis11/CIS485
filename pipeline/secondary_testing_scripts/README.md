# Secondary Testing Scripts (`pipeline/secondary_testing_scripts/`)

This directory holds a suite of standalone scripts to evaluate portions of the Computer Vision pipeline sequentially. Rather than integrating everything back into `main_realtime.py` or sending it all sequentially to the Streamlit app, these testers execute narrow, focused tasks on raw footage to diagnose model accuracy constraints independently.

## Testing Methodologies Present

* **`keypoint_detection_only.py`**: Ignores ByteTrack players completely. Processes a hardcoded video specifically looking to render pitch landmarks and output an overlaid `.mp4` for validation of the `yolo8n-pose` weights.
* **`player_tracker_only.py`**: Mutes field homography steps and team classifiers. Strictly highlights bounding boxes and UUID assignments generated by ByteTrack on YOLO player tracking outputs.
* **`fast_transcode.py`**: A quick video utility to compress, scale, or modify fps of `raw/` files rapidly before running expensive inference against them.
* **`test_full_image.py` / `test_keypoints.py`**: Designed to be unit tested against static `.jpg` images from Roboflow or SoccerNet to accurately measure precision/recall and generate side-by-side matrices without dealing with temporal logic.

### Usage
Scripts in this folder are intended to be executed from the CLI manually. Relative path resolutions may require running these tests from the project root.

```bash
# Example execution:
python pipeline/secondary_testing_scripts/keypoint_detection_only.py
```
