Project requirements:
-Python 3.13
-streamlit, or another suggested web app framework

Goals:
Page 1:
-Create a web app that allows for users to upload a short video of a football match
-Process said video using the player-tracking yolo model and the key point detection model
-Display the annotated frames in a user-friendly way
-Allow users to download the annotated video

Page2:
-Show a demonstration of the player-tracking yolo model and the key point detection model from a pre-processed video
-Allow users to download these videos

Page3:
-Prepare this page for allowing users to upload a video of a football match
-Then run the player-tracking yolo model and the key point detection model on the uploaded video
-Functionallity will come in the future for displaying the video game like rendition of the annoated video

The web application should be simple and modern in design. It should be easy to navigate and use. It should be designed in a manner that is simple to change and update in the future
